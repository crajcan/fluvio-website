Next you will have to add Fluvio to your $PATH environment variable. There are three main shells that see common usage: Fish, Bash, and Zsh. Each one uses a different method to edit $PATH.

If you are unsure of which shell you are using, the following command will tell you:

$ echo $0
/bin/zsh

Once you know which shell you are using, use one of the following commands to modify your $PATH.

Fish
$ fish_add_path ~/.fluvio/bin
Bash
$ echo 'export PATH="$PATH:$HOME/.fluvio/bin"' >> .bashrc
Zsh
$ echo 'export PATH="$PATH:~/.fluvio/bin"' >> .zshrc
